Basic HTML

HTML bestaat uit verschillende elementen, elementen bestaan uit 3 delen, de opening, de content en de sluiting

Om een element te maken moet je beginnen met <p> in de binnenkant van de <> bepaal je welk element je maakt

Om een element te sluiten moet je aan het einde </p> toevoegen met weer in de binnenkant van de <> de het element naam met een / ervoor voor sluiten

Dus om een p element te maken om tekst te laten zien zou je <p>tekst in het p element</p> plaatsen

Basic css

CSS gebruik je om attributen op HTML elementen te veranderen.

Om dat te doen heb je in CSS selectors, selectors zijn: element, class en id

Voor class en id moet je in HTML elementen een extra waarde toevoegen

dit doe je binnen de opening van het element doormiddel van class="" en id=""

het id waarde moet uniek zijn en kan maar op een element, de class waarde kan op meerdere elementen.


Om met deze waardes in CSS de elementen te selecteren heb je verschillende manieren maar de belangerijkste zijn dit:

p {} selecteert alle p elementen in je HTML

.class {} selecteert alle elementen met een class en begint met een .

#id {} selecteert het element met het id en begint met een #

Tekst

Tekst elementen zijn simpele elementen die alleen tekst laten zien.

Er zijn voornamelijk 3 soorten tekst elementen: headers, paragraphs, span en links


Zie hieronder voor de soorten tekst elementen:

<h1> Header 1

<h2> Header 2

<h3> Header 3

<h4> Header 4

<h5> Header 5
<h6> Header 6

<p> Paragraph

<span> Span

<a href="#"> linkjes

Span elementen zijn speciale elementen die je in het midden van je tekst kan toevoegen om een stuk van je tekst te stylen

Bijvoorbeeld <span class="voorbeeld">Ik zit in een span!</span> wordt Ik zit in een span! zonder dat er extra tekst elementen toe te voegen

Linkjes

Linkjes bestaan voornamelijk uit maar 1 element maar er zijn wel veel manieren om het uiterlijk van linkjes te veranderen met behulp van CSS

Linkjest bestaan uit <a> elementen met een belangrijk verschil van normale tekst elementen.

<a> elementen hebben een verplichte extra waarde nodig genaamd de href

De href waarde is de link waarnaar de pagine wordt veranderd als je erop klikt

De href waarde kan je bepalen doormiddel van href="" binnen <a> dus <a href="#">

De tekst binnen <a href="#"></a> is de tekst die zichtbaar zal zijn voor de gebruiker

Bijvoorbeeld: <a href="https://youtube.com">Youtube</a> word Youtube


Standaard zijn linkjes blauw met een underline maar dat kan je met behulp van CSS veranderen

Om de kleur van linkjes te veranderen gebruik je het color attribuut

Om de underline van linkjes te verwijderen/veranderen gebruik je het text-decoration attribuut

Groepen

Groepen zijn gemaakt met maar 1 element maar kan oneindig element in zich hebben.

Groepen betaan uit <div> elementen en zijn belangrijk voor de opmaak van een website

<div> Worden voornamelijk gebruikt om meerdere elementen in groepen te verdelen

<div> Zoals alle elementen kan de <div> een class en id hebben
