In dit voorbeeld zullen we uitleggen hoe websites werken met gebruik van html en css.

Ook kan je zelf html en css schrijven en het resultaat zien veranderen.

Html bestaat voornamelijk uit "elementen" dit zijn objecten die iets laten zien op de web pagina.

er zijn verschillende soorten elementen voornamelijk tekst, knoppen, groepen en formulieren

Tekst elementen zijn elementen zoals h1, h2, h3, etc voor verschillende soorten headers, p voor paragraven en a for links naar andere paginas/websites


Voor meer uitleg over element hebben we een uitleg voorberijd die je kan gebruiken om te leren hoe ze werken.
